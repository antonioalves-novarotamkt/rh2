## 🔧 Personalizações Fáceis

### Mudar Cores Globais

No arquivo HTML, procure por `:root` no CSS:

```css
:root {
    --primary-blue: #2C5FA3;      /* Azul principal */
    --primary-gold: #D4A574;      /* Dourado */
    --dark-wine: #6B3E4F;         /* Marrom/Vinho */
}
```

### Mudar Textos

Basta procurar o texto no HTML e substituir.

### Adicionar Redes Sociais

No footer, procure por "Conecte-se" e adicione seus links.

## 🌐 Colocar Online (Hospedagem Grátis)

### Opção 1: **Vercel** (Recomendado - SUPER FÁCIL)
1. Vá para [vercel.com](https://vercel.com)
2. Clique em "New Project"
3. Selecione "Import Git Repository" OU arraste sua pasta
4. Em 2 segundos seu site estará online!
5. Você ganha um domínio tipo: `seu-projeto.vercel.app`

### Opção 2: **Netlify**
1. Vá para [netlify.com](https://netlify.com)
2. Faça login
3. Arraste sua pasta
4. Pronto! Site online em segundos

### Opção 3: **GitHub Pages** (Profissional)
1. Crie um repositório no GitHub chamado `seu-usuario.github.io`
2. Envie seus arquivos
3. Seu site estará em `seu-usuario.github.io`
